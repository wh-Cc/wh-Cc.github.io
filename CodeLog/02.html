在项目中统一样式 ​

在 CSS 中统一应用样式保持风格统一

1. CSS 变量（Custom Properties） ​

CSS 变量允许你在文档中定义可复用的值。你可以在 :root 或其他选择器上设置变量，然后在任何地方使用它们。

css
:root {
  --primary-color: #3498db;
  --font-stack: 'Helvetica Neue', sans-serif;
}

body {
  font-family: var(--font-stack);
}

button.primary {
  background-color: var(--primary-color);
}

2. 使用unocss 统一样式 ​

在你的项目中创建一个 Unocss 配置文件，例如 unocss.config.js。

javascript
import { defineConfig } from 'unocss';

export default defineConfig({
  theme: {
    colors: {
        primary: {
            light: '#3498db',
            dark: '#2980b9',
        },
        secondary: {
            light: '#f1c40f',
            dark: '#f39c12',
        },
        // 定义更多颜色...
    },
  },
  // 其他配置...
});

3.使用tailwind ​

tailwind.config.js

javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: {
          light: '#3498db',
          DEFAULT: '#2980b9', // 默认值用于 'primary'
          dark: '#2471a3',
        },
        secondary: {
          light: '#f1c40f',
          DEFAULT: '#f39c12',
          dark: '#e08e0b',
        },
        // 添加更多颜色...
      },
      fontFamily: {
        sans: ['"Helvetica Neue"', 'sans-serif'],
        serif: ['"Times New Roman"', 'serif'],
        // 添加更多字体族...
      },
    },
  },
  // 其他配置...
}

Element UI 2.x ​

css
/* 在你的 global.css 或样式标签中定义变量 */
:root {
  --el-color-primary: #409eff; /* Element UI 的默认主题色 */
}

/* 覆盖默认的主题色 */
.el-button--primary {
  background-color: var(--el-color-primary);
  border-color: var(--el-color-primary);
}

Element Plus ​

Element Plus 允许你通过配置文件来修改主题色：

javascript
// main.js 或其他入口文件
import { createApp } from 'vue';
import ElementPlus from 'element-plus';
import 'element-plus/dist/index.css';

const app = createApp(App);

app.use(ElementPlus, {
  size: 'medium', // 设置 Element Plus 的默认尺寸
  zIndex: 2000, // 设置 Element Plus 的默认 z-index
  // 其他配置...
});

然后，在你的 CSS 文件中覆盖颜色变量：

css
:root {
  --el-color-primary: #你选择的颜色;
}

prefers-color-scheme ​

prefers-color-scheme CSS 媒体特性用于检测用户是否有将系统的主题色设置为亮色或者暗色

css

.day {
  background: #eee;
  color: black;
}
.night {
  background: #333;
  color: white;
}


@media (prefers-color-scheme: dark) {
  .day.dark-scheme {
    background: #333;
    color: white;
  }
  .night.dark-scheme {
    background: black;
    color: #ddd;
  }
}

@media (prefers-color-scheme: light) {
  .day.light-scheme {
    background: white;
    color: #555;
  }
  .night.light-scheme {
    background: #eee;
    color: black;
  }
}
