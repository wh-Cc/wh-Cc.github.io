On this page

在 Vue 3 中，使用进阶特性可以创建功能丰富的应用程序。以下是一些，展示了如何利用 Vue 3 的进阶特性：

1. 自定义指令： ​

自定义指令可以用于实现复杂的交互逻辑，比如自定义的拖拽指令。

javascript
const draggable = {
  mounted(el) {
    el.draggable = true;
    el.addEventListener('dragstart', () => {
      // 处理拖拽开始的逻辑
    });
  }
};

2. 服务器端渲染 (SSR)： ​

SSR 可以用于提高应用的首屏加载速度和 SEO。

javascript
const { createApp } = require('vue');
const { renderToString } = require('@vue/server-renderer');

const app = createApp(App);

// 在服务器端渲染应用
renderToString(app).then(html => {
  // 发送 HTML 到客户端
});

3. 动态组件和vnode： ​

动态组件可以根据条件渲染不同的组件。

html
<component :is="currentComponent"></component>

4. 组件插槽(v-slots)： ​

插槽允许你将内容注入组件内部。

html
<template v-slot:header="slotProps">
  <h1>{{ slotProps.headerText }}</h1>
</template>

5. 异步组件： ​

异步组件可以用于按需加载。

javascript
const AsyncComponent = () => ({
  // 需要加载的组件
  component: import('./components/AsyncComponent.vue'),
  // 组件加载时显示的占位符
  loading: LoadingComponent,
  // 加载失败时显示的组件
  error: ErrorComponent,
  // 延迟时间，单位毫秒
  delay: 200,
});

6. provide和injectAPI： ​

跨组件传递数据。

javascript
// 祖先组件中提供数据
provide('theme', 'dark');

// 后代组件中注入数据
inject('theme', 'light');

7. teleport组件： ​

将组件的 DOM 移动到 DOM 的另一个位置。

html
<teleport to="#modal">
  <div>
    <!-- 模态框内容 -->
  </div>
</teleport>

8. 响应式系统的readonly和shallowReactive： ​

创建只读或浅层响应式对象。

javascript
const original = { prop: 'value' };
const readonlyProxy = readonly(original);

const shallowReactiveObject = shallowReactive({ nested: { prop: 'value' } });

9. 错误处理： ​

全局错误处理器。

javascript
app.config.errorHandler = (err, vm, info) => {
  // 处理错误
};

10. 自定义渲染器： ​

创建自定义渲染器。

```javascript
const renderer = {
  render: (vnode, container) => {
    // 自定义渲染逻辑
  }
};
```

11. TypeScript 支持： ​

使用 TypeScript 定义组件。

```typescript
import { defineComponent, ref } from 'vue';

interface Props {
  msg: string;
}

const MyComponent = defineComponent<Props>({
  setup(props) {
    const count = ref(0);
    return { count };
  }
});
```

12. 性能优化： ​

使用生产模式构建应用。

```shell
vue build --prod
```

13. 模块化： ​

按需引入 Vue 3 的部分。

```javascript
import { createApp } from 'vue';
import { router } from './router';
// 只引入需要的部分
```

14. <script setup>语法糖： ​

简化 `setup` 函数的书写。

```javascript
<script setup>
import { ref } from 'vue';

const count = ref(0);
</script>
```

15. 全局 API 的自定义封装： ​

创建自定义的插件。

```javascript
app.config.globalProperties.$myCustomFunction = () => {
  // 自定义函数逻辑
};
```

16. 使用defineAsyncComponent定义异步组件： ​

定义按需加载的组件。

```javascript
const MyAsyncComponent = defineAsyncComponent(() =>
  import('./components/MyComponent.vue')
);
```

这些展示了 Vue 3 进阶特性的多样性和强大功能，可以帮助开发者构建高效、可维护的应用程序。
