On this page

Vue 3 的 Composition API 是一种新的组件编写方式，它提供了一种更灵活、更模块化的方法来组织组件逻辑。Composition API 允许你将组件的逻辑分散到不同的函数中，使得代码更加清晰和易于维护。以下是 Composition API 的一些核心概念和用法：

1. setup 函数 ​

  • setup 是 Composition API 的入口点，它在组件创建之前执行。它接收 props、context 作为参数，返回一个对象，该对象中的属性可以在模板中使用。
javascript
import { ref, onMounted } from 'vue';

export default {
  setup(props, { slots, emit }) {
    const count = ref(0);

    onMounted(() => {
      console.log('Component is mounted!');
    });

    return {
      count
    };
  }
};

2. 响应式状态 ​

  • 使用 ref 创建基本的响应式引用，适用于基本数据类型。
  • 使用 reactive 创建复杂的响应式状态，适用于对象。
javascript
import { ref, reactive } from 'vue';

const count = ref(0);
const state = reactive({
  name: 'Vue 3',
  likes: 100
});

3. 计算属性 ​

  • 使用 computed 创建计算属性，它们是基于它们的响应式依赖进行缓存的。
javascript
import { computed } from 'vue';

const doubledCount = computed(() => count.value * 2);

4. 侦听器 ​

  • 使用 watch 来侦听响应式引用或状态的变化，并在变化发生时执行回调。
javascript
import { watch } from 'vue';

watch(count, (newValue, oldValue) => {
  console.log(`count changed from ${oldValue} to ${newValue}`);
});

5. 生命周期钩子 ​

  • Vue 3 提供了 onMounted、onUpdated、onUnmounted 等生命周期钩子函数。
javascript
import { onMounted, onUpdated, onUnmounted } from 'vue';

onMounted(() => {
  // 组件挂载后执行
});

onUpdated(() => {
  // 组件更新后执行
});

onUnmounted(() => {
  // 组件卸载前执行
});

6. 事件处理 ​

  • 在 setup 函数中，你可以定义事件处理函数，并在返回对象中暴露它们，以便在模板中使用。
javascript
function increment() {
  count.value++;
}

return {
  count,
  increment
};

7. 提供/注入 API ​

  • 使用 provide 和 inject 来实现跨组件的状态共享。
javascript
import { provide, inject } from 'vue';

provide('theme', 'dark');

// 在另一个组件中
const theme = inject('theme');

8. 自定义 Hook ​

  • 可以编写自定义 Hook 来封装和复用逻辑。
javascript
// useCounter.js
export function useCounter() {
  const count = ref(0);
  const increment = () => {
    count.value++;
  };
  return { count, increment };
}

示例组件 ​

以下是一个使用 Composition API 的 Vue 3 组件示例：

vue
<template>
  <div>
    <h1>{{ state.name }} has {{ state.likes }} likes</h1>
    <p>{{ doubledCount }}</p>
    <button @click="increment">Like</button>
  </div>
</template>

<script setup>
import { ref, reactive, computed, watch, onMounted } from 'vue';
import useCounter from './useCounter.js';

const { count, increment } = useCounter();

const state = reactive({
  name: 'Vue 3',
  likes: 0
});

const doubledCount = computed(() => state.likes * 2);

watch(count, (newValue, oldValue) => {
  state.likes = newValue;
});

onMounted(() => {
  console.log('Component is mounted!');
});
</script>

这个示例展示了如何使用 setup 函数、自定义 Hook、响应式状态、计算属性、侦听器和生命周期钩子。Composition API 提供了一种强大的方式来组织组件逻辑，使得代码更加模块化和易于维护。
