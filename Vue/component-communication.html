在 Vue 3 中，Pinia 是官方推荐的存储库，用于替代 Vuex 作为状态管理库。Pinia 提供了一个更简单、更类型安全的方式来管理应用的状态。以下是 Vue 3 组件通信方式，包括 Pinia 的示例：

1. Props 和 Events (父子通信) ​

  • 父组件通过 Props 传递数据给子组件。
  • 子组件通过 Emits 触发事件来通知父组件。

父组件:

vue
<template>
  <ChildComponent :parentMessage="message" @child-emits="handleChildEvent" />
</template>

<script setup>
import { ref } from 'vue';
import ChildComponent from './ChildComponent.vue';

const message = ref('Hello from parent!');

function handleChildEvent(data) {
  console.log('Received from child:', data);
}
</script>

子组件:

vue
<template>
  <div>
    <p>{{ parentMessage }}</p>
    <button @click="emitMessage">Send to Parent</button>
  </div>
</template>

<script setup>
defineProps(['parentMessage']);
defineEmits(['child-emits']);

function emitMessage() {
  emit('child-emits', 'Hello from child!');
}
</script>

2. provide / inject (跨组件实例通信) ​

  • 祖先组件通过 provide 向所有后代组件提供数据。
  • 后代组件通过 inject 来接收数据。

祖先组件:

vue
<template>
  <div>
    <ChildComponent />
  </div>
</template>

<script setup>
import { provide } from 'vue';
import ChildComponent from './ChildComponent.vue';

provide('theme', 'dark');
</script>

后代组件:

vue
<script setup>
import { inject } from 'vue';

const theme = inject('theme');
</script>

3. Event Bus (非官方，但常用) ​

  • 使用一个空的 Vue 实例作为中央事件总线。

创建 Event Bus:

javascript
// eventBus.js
import { reactive, readonly } from 'vue';
export default readonly(reactive(new Vue()));

使用 Event Bus:

vue
<script setup>
import eventBus from './eventBus.js';

// 监听事件
watchEffect(() => {
  console.log('The theme is:', eventBus.theme);
});

// 触发事件
eventBus.$emit('themeChange', 'light');
</script>

4. Pinia (或vuex、vue-router 等第三方库也可以帮助实现组件通讯) ​

  • 使用 Pinia 进行全局状态管理。

安装 Pinia:

bash
npm install pinia@next

创建 Store:

javascript
// useCounter.js
import { defineStore } from 'pinia';

export const useCounterStore = defineStore('counter', {
  state: () => ({
    count: 0
  }),
  actions: {
    increment() {
      this.count++;
    }
  }
});

在组件中使用:

vue
<script setup>
import { useCounterStore } from './useCounter.js';

const counterStore = useCounterStore();

function increment() {
  counterStore.increment();
}
</script>

5. 响应式系统 ​

  • 使用 Vue 的响应式 API 来创建跨组件的状态和逻辑。

共享状态:

javascript
// sharedState.js
import { reactive } from 'vue';

export default reactive({
  count: 0
});

组件使用:

vue
<script setup>
import sharedState from './sharedState.js';

watchEffect(() => {
  console.log(sharedState.count);
});
</script>
