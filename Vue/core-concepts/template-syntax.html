Vue 的模板语法允许你在 HTML 模板中声明性地描述用户界面。模板基于 HTML，通过 Vue 提供的特殊语法扩展，可以绑定到组件的数据上。以下是 Vue 模板语法的一些核心知识点：

1. 插值表达式 ​

使用双花括号 将 JavaScript 表达式插入到文本或 HTML 中。

vue
<p>Message: {{ message }}</p>

2. 指令 ​

Vue 的指令以 v- 前缀开头，用于在元素上应用动态行为。

  • v-bind: 动态地绑定一个或多个属性，或一个组件 prop 到表达式。

    vue
    <img v-bind:src="imageSrc" alt="Image">

  • v-model: 在表单输入和应用状态之间创建双向数据绑定。

    vue
    <input v-model="username">

  • v-on: 监听 DOM 事件或组件事件。

    vue
    <button v-on:click="submit">Submit</button>

  • v-for: 基于源数据多次渲染一个元素或模板块。

    vue
    <ul>
  <li v-for="item in items" :key="item.id">{{ item.text }}</li>
</ul>

  • v-if, v-else-if, v-else: 条件性地渲染元素。

    vue
    <p v-if="user.isLoggedIn">欢迎回来, {{ user.name }}!</p>
<p v-else>请登录。</p>

  • v-show: 根据条件切换元素的 CSS 显示属性。

    vue
    <div v-show="isVisible">可见条件满足时显示</div>

  • v-cloak: 与 CSS 结合使用，防止 Vue 插值表达式在页面上闪烁。

    vue
    <style>
[v-cloak] { display: none; }
</style>
<div v-cloak>{{ message }}</div>

3. 指令修饰符 ​

修饰符提供了一种方法，可以在使用指令时执行额外的操作。

  • .prevent: 调用事件的 event.preventDefault() 方法。

    vue
    <button v-on:click.prevent="submit">Submit</button>

  • .stop: 调用事件的 event.stopPropagation() 方法。

    vue
    <div v-on:click.stop="handleClick">Don't click me!</div>

  • .self: 确保事件是从该元素本身触发的。

    vue
    <div v-on:click.self="handleClick">Click me!</div>

4. 动态指令 ​

使用方括号 [ ] 动态地绑定指令名称。

vue
<button v-bind:[dynamicAttr]="value">Click me</button>

5. 作用域插槽 ​

允许将内容分发到组件的特定部分。

vue
<template v-slot:header="slotProps">
  <h1>{{ slotProps.headerText }}</h1>
</template>

6. 模板中的表达式 ​

模板中的表达式限制在一个单独的 JavaScript 语句中，不支持像 if 或 for 这样的 JavaScript 语句。

7. 组件 ​

自定义的可复用组件，可以在模板中使用。

vue
<MyComponent :key="uniqueId" :propA="someValue"></MyComponent>

示例 ​

以下是一个简单的 Vue 组件模板示例，展示了模板语法的使用：

vue
<template>
  <div>
    <h1>{{ title }}</h1>
    <p v-if="showDescription">这里是一些描述。</p>
    <button v-on:click="toggleDescription">切换描述</button>
    <ul>
      <li v-for="(item, index) in items" :key="index">
        {{ item.name }} - {{ item.detail }}
      </li>
    </ul>
    <input v-model="message">
  </div>
</template>

<script>
export default {
  data() {
    return {
      title: '示例组件',
      showDescription: true,
      items: [
        { name: '项目 1', detail: '详情 1' },
        { name: '项目 2', detail: '详情 2' }
      ],
      message: ''
    };
  },
  methods: {
    toggleDescription() {
      this.showDescription = !this.showDescription;
    }
  }
};
</script>

这个示例展示了插值表达式、条件渲染、列表渲染、事件处理和双向数据绑定。Vue 的模板语法是声明式的，它使得开发者能够以一种非常直观的方式来构建动态的用户界面。
