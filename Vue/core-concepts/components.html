Vue 3 组件是构建 Vue 应用程序的基本单元。以下是 Vue 3 组件的一些核心知识点和示例：

1. 组件定义 ​

在 Vue 3 中，你可以使用 defineComponent 函数或 <script setup> 语法来定义组件。

  • 使用 defineComponent:

    vue
    import { defineComponent } from 'vue';

export default defineComponent({
  name: 'MyComponent',
  // 组件选项...
});

  • 使用 <script setup>:

    vue
    <script setup>
import { ref } from 'vue';

const count = ref(0);
</script>

<template>
  <div>{{ count }}</div>
</template>

2. Props 和 Emits ​

  • Props: 父组件传递给子组件的数据。
  • Emits: 子组件触发的事件，供父组件监听。
vue
<script setup>
defineProps({
  initialCount: Number
});

defineEmits(['updateCount']);
</script>

3. 响应式状态 ​

使用 reactive 或 ref 来声明组件内的响应式状态。

javascript
import { reactive, ref } from 'vue';

const state = reactive({
  count: 0
});

const countRef = ref(0);

4. 模板语法 ​

Vue 的模板语法允许你声明式地将组件的状态渲染进 DOM。

vue
<template>
  <div>{{ state.count }}</div>
</template>

5. 事件处理 ​

使用 v-on 或 @ 缩写来监听 DOM 事件或组件事件。

vue
<template>
  <button @click="increment">Increment</button>
</template>

<script setup>
function increment() {
  state.count++;
}
</script>

6. 计算属性 ​

使用 computed 来声明依赖于响应式状态的派生状态。

javascript
import { computed } from 'vue';

const doubledCount = computed(() => state.count * 2);

7. 侦听器和副作用 ​

使用 watch 和 watchEffect 来侦听响应式状态的变化，并执行副作用。

javascript
import { watch, watchEffect } from 'vue';

watchEffect(() => {
  console.log(state.count);
});

watch(countRef, (newValue, oldValue) => {
  console.log(`count changed from ${oldValue} to ${newValue}`);
});

8. 生命周期钩子 ​

Vue 3 组件具有多个生命周期钩子，如 onMounted, onUpdated, onUnmounted 等。

javascript
import { onMounted } from 'vue';

onMounted(() => {
  console.log('Component is mounted');
});

9. 组件通信 ​

使用 Props、Emits、Event Bus、Vuex 或 provide/inject 来实现组件间的通信。

10. 自定义指令 ​

Vue 3 允许你创建自定义指令来扩展 Vue 的功能。

javascript
import { directive } from 'vue';

const myDirective = directive('my-directive', {
  // 指令的选项...
});

示例组件 ​

下面是一个包含上述核心知识的简单 Vue 3 组件示例：

vue
<template>
  <div>
    <h1>{{ title }}</h1>
    <p>{{ state.count }}</p>
    <button @click="increment">Increment</button>
  </div>
</template>

<script setup>
import { ref, reactive, computed, watchEffect, onMounted } from 'vue';

const props = defineProps({
  title: String
});

const state = reactive({
  count: props.initialCount || 0
});

const doubledCount = computed(() => state.count * 2);

watchEffect(() => {
  console.log(`The count is now: ${state.count}`);
});

function increment() {
  state.count++;
}

onMounted(() => {
  console.log('Component is mounted!');
});
</script>

这个示例展示了如何定义 Props、响应式状态、计算属性、侦听器、生命周期钩子，以及如何在模板中使用它们。这是 Vue 3 组件开发的基础。
